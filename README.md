# 闲鱼(Goofish) 深度选品分析半自动抓取系统

## 📌 项目简介
本项目是一个专门针对中国 🇨🇳 闲鱼（Goofish）二手电商平台开发的半自动、深度数据抓取与选品分析工具。系统基于 Python 语言与 **Playwright** 自动化框架，通过两阶段分布式深度爬取技术，高精度提取闲鱼详情页的核心互动数据，为电商卖家提供直观的爆款数据决策支持。

---

## 🎯 解决什么问题
1. **列表页信息截断与失真**：闲鱼列表页中的商品标题常因前台排版被省略号大量截断。本项目直接深入详情页，读取完整的**网页原生标题**作为纯净准确的商品名称。
2. **核心选品指标缺失**：搜索列表页不提供“浏览量”。本项目通过二级详情深度解析，抓取每个商品的精准“想要数”与“浏览数”，首创**“想要/浏览”转化率算法**，有效剔除僵尸曝光，筛选高意向度的真爆款。
3. **极高频的反爬虫滑块拦截**：常规开源爬虫或注入式自动化脚本进入阿里系产品时，会频繁触发滑块且无限报错 `error:mw5t4`。本项目打通指纹浏览器生态，实现真正无痕的纯净无感知数据采集。

---

## 🧱 技术架构
* **核心核心语言**：`Python 3.x`
* **自动化控制框架**：`Playwright` (sync_api 同步框架)
* **指纹环境及防关联网络**：`AdsPower 指纹浏览器`
* **通讯协议**：`CDP (Chrome DevTools Protocol)` 远程调试协议 & WebSocket
* **数据清洗与导出**：`Re` (全局正则表达式匹配)、`CSV` (内置多字节安全写入)

---

## ✨ 功能特性
* **环境全自动唤醒**：无须人工在指纹浏览器客户端手动点击，脚本通过本地 API 自动拉起并接管 AdsPower 内指定的独立指纹窗口。
* **二级详情深度挖掘**：采用两阶段采集逻辑。第一阶段自动化检索列表并安全抽取去重商品详情 URL；第二阶段批量循环进入详情页解析全量原生字段。
* **高价值多维度字段解析**：
  * 商品唯一 ID、**商品名称（100%还原网页标题）**
  * 精准商品价格、想要人数、浏览次数
  * **卖家真实昵称**、商品直达链接
* **高潜力爆款自动排序**：系统自动计算 `转化率 = 想要数 / 浏览数` 权重，并根据该权重对所有生成数据进行**降序排列**，核心爆款一秒锁定。
* **Excel 友好型输出**：原生支持 `utf-8-sig` 编码写入 CSV，确保直接使用 Microsoft Excel 打开时绝不出现乱码。

## 🚀 如何运行

### 1. 环境准备
确保你的本地 Python 环境已安装对应依赖：
```bash
pip install playwright requests
```
### 2. 配置 AdsPower 接口

1. 打开 AdsPower 客户端，进入 **“设置” -> “本地设置” -> “应用接口”**。
2. 确认接口处于开启状态，并检查记录你的**内置接口端口**（例如：`50325`）。
3. 如果客户端默认开启了 API Key 认证，请在设置中手动将其**关闭**（或确保采用基于 user_id/serial_number 的免验证接口）。

### 3. 修改代码参数

打开 Python 脚本，修改顶部的参数区域：

```python
KEYWORDS = "你的目标选品关键词"
MAX_PAGES = 3  # 目标搜集的列表页数
ADS_API_PORT = 50325  # 对应你的 AdsPower 端口
ENV_SERIAL_NUMBER = "1"  # 对应你在 AdsPower 中的账号序号（如列表第一个填 1）

```

### 4. 执行脚本

确保 AdsPower 客户端在后台处于登录运行状态，在控制台执行：

```bash
python main.py
```

*脚本会自动唤醒对应的指纹窗口，并在窗口中全自动完成检索、下拉刷新和详情页的连续数据读取。*

## 💡 技术亮点

- **CDP 协议无痕接管**：不同于在 Playwright 或 Selenium 中频繁通过 `add_init_script` 强行覆盖 `navigator.webdriver` 变量（容易被安全脚本秒破），本项目直接基于 CDP 协议控制 100% 真实的独立指纹浏览器环境，在底层彻底消除爬虫特征。
- **属性前缀模糊匹配**：闲鱼前端类名高度混淆（如 `price--OEWLbcxC`），后缀带有高频变化的随机干扰码。本项目全面采用 `div[class^="price--"]` 开头包含匹配语法，具备极强的抗前端重构能力，大大延长了代码的生命周期。
- **仿人化随机对抗等待**：列表懒加载滑动、详情页跳转之间加入了科学波动的缓冲延时，完美模拟人类用户的正常浏览节奏，防封抗拉能力达到企业级标准。

## ⚠️ 遇到的问题与解决方案

- **问题：手动划过验证码后仍无限提示验证失败（Error: mw5t4）**
  - *原因*：阿里安全风控系统检测到常规自动化脚本底层的控制流二进制痕迹，无论滑块如何正确滑动，后台均拒绝通过认证。
  - *解决方案*：弃用本地伪装机制，全面重构为 CDP 模式连接外部 AdsPower 指纹环境，利用指纹浏览器原生隔离环境和纯净硬件指纹，直接绕过防爬检测链。
- **问题：列表页抓取的标题信息不全，且数据维度不足**
  - *原因*：列表页前端组件出于排版和响应性能考虑，对商品文字进行了动态截断，且未下发浏览量等高阶互动数据。
  - *解决方案*：升级为两阶段爬虫架构，将首屏作为链接孵化器，全量数据交由深度解析详情页完成，保证数据准确率达 100%。
